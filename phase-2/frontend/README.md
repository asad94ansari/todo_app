# Todo Frontend

This is the frontend for the Todo application, built with Next.js and TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in a `.env.local` file:
```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- User authentication flow (login/signup)
- Todo dashboard with CRUD operations
- Real-time task completion toggling
- Responsive UI with Tailwind CSS
- Secure session management
- Optimistic UI updates
- Error handling and loading states
